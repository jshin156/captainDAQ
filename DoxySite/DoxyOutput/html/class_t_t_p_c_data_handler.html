TTPCDataHandler::WiresRMS TTPCDataHandler::GetWiresRMS ( ) const

Calculate the voltage noise RMS for all wires. This makes two passes: one pass calculates an initial voltage pedestal (mean0) and noise RMS (RMS0). The second pass recalculates a new pedestal (mean1) and RMS (RMS1) excluding bins deviating by more than 3*RMS0 from mean0. RMS1 is the final result.
